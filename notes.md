### bit representation

| piece      | bit pattern |
| ---------- | ----------- |
| empty cell | 000         |
| pawn       | 001         |
| knight     | 010         |
| bishop     | 011         |
| rook       | 100         |
| queen      | 101         |
| king       | 110         |
| (unused)   | 111         |

| team  | bit value |
| ----- | --------- |
| white | 0         |
| black | 1         |

\[piece (3 bits)\]\[team (1 bit)\]
